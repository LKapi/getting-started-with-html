HTML
- stands for hypertext markup language
- it is made up of:
text content,
markup (what it looks like),
references to other documents, and
links to other pages.


Formatting information:
b - will bold text
em - will italicise text

Attributes:
 ATTR = "value"
anchor (a) - is used for making links
use (a href="content") (/a)

Images:
 img
use (img src = "uri" alt - "text") --- with void tag at end

Line Break:
(br) - void ending
It is inline - ends the line
Applies only to text

Paragraph:
(p) content (/p) -- no void tag
Makes an invisible box around the text

The following are Inline:
a
span
br
img
strong

The following are Block
div
p
form
